## 欢迎使用DramKit

DramKit是一个工具箱，名字中的Dram取自哆啦A梦的英文名Doraemon。DramKit的目标是提升日常工作效率。如果您的工作或研究方向跟量化、金融、数据分析等领域相关，且使用Python编码，那么DramKit可能会对您有用。

DramKit的主要工具包括：日志记录、绘图、文件读写、统计分析，以及其他一些有趣的算法。

DramKit的内容和文档将不断完善。

安装命令：

```
pip install dramkit --upgrade
```
